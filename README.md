# 🔥 FLAMES Score Love Game

A fun name-based compatibility calculator built with **Flutter**.

**FLAMES** stands for: **F**riends · **L**overs · **A**ffection · **M**arriage · **E**nemy · **S**iblings

## Preview

Enter two names → cancel matching letters → count remaining → eliminate from FLAMES → discover your score!

| Letter | Meaning    | Emoji |
|--------|------------|-------|
| F      | Friends    | 🤝    |
| L      | Lovers     | 💕    |
| A      | Affection  | 🥰    |
| M      | Marriage   | 💍    |
| E      | Enemy      | ⚔️    |
| S      | Siblings   | 👫    |

## Tech

- **Flutter** 3.6+ with Material 3 Design
- Cross-platform: Android, iOS, Web, Desktop
- Clean architecture: models / screens / widgets
