# FLAMES Love Game — Recommendations

> Generated: June 22, 2026
> Status: Project is 100% feature-complete per Roadmap

## Priority Recommendations

### 1. 🚀 Publish to Google Play Store
The AAB build is ready. Screenshots exist under `assets/store_screenshots/`.
Only need a Play Console account ($25 one-time). Light AdMob integration
could generate passive revenue, or keep it free as a portfolio piece.

### 2. 📊 Percentage Mode
Add a compatibility percentage alongside the classic FLAMES letter result
(e.g., "78% Match"). More intuitive for casual users and more shareable
on social media. Algorithm: map remaining letter count to a 0-100 scale.

### 3. 📅 Daily FLAMES
A daily prompt where all users get the same random celebrity or default
pair. Show global result stats via Firebase. Drives daily engagement,
retention, and social sharing.

### 4. 🎭 Couple Avatars / Themes
Let users pick emoji avatars for each name. Save recent couples with
cute nicknames. Makes the result card more personal and shareable.

### 5. 📊 Analytics & Crash Reporting
Add Firebase Crashlytics + Analytics. Understand real usage patterns
(calculate once? share? replay?) before investing in v2 features.

### 6. 📱 iOS Widget / Android Shortcut
Home screen widget for quick FLAMES check without opening the app.
High novelty value, viral-ready.

### 7. 🏷️ Rebrand for App Store
Current name is descriptive but long. Options: "Love Spark", "Crusher",
"Heart Check", "Flame Match". Shorter names rank better in app store search.

## Backburner
