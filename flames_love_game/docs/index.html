FLAMES

Love Score Game

💕 What is FLAMES?

FLAMES is a fun name-based compatibility game. Enter two names, and the app calculates your score — from Friends (🤝) to Marriage (💍).

🤝 F Friends
💕 L Lovers
🥰 A Affection
💍 M Marriage
⚔️ E Enemy
👫 S Siblings

⚙️ How It Works

Step 1 — Enter Two Names

Type in two names (e.g., "Alchie" and "Tagudin").

Step 2 — Cancel Matching Letters

Each common letter cancels out. Only the uncancelled letters remain:

Name 1: ALCHIE  →  A:1 L:1 C:1 H:1 I:1 E:1
Name 2: TAGUDIN →  T:1 A:1 G:1 U:1 D:1 I:1 N:1

Cancelled: A, I
Remaining: LCHE + TGUDN = 9 letters

Step 3 — Count & Eliminate

Use the count (9) to remove letters from FLAMES one by one:

FLAMES  →  6 letters
(9-1) % 6 = 2  →  remove 'A'  →  FLMES

FLMES   →  5 letters
(9-1) % 5 = 3  →  remove 'E'  →  FLMS

FLMS    →  4 letters
(9-1) % 4 = 0  →  remove 'F'  →  LMS

LMS     →  3 letters
(9-1) % 3 = 2  →  remove 'S'  →  LM

LM      →  2 letters
(9-1) % 2 = 0  →  remove 'L'  →  M

Result: M → Marriage 💍
